HackYourFuture Alumni Platform

HackYourFuture(HYF) Alumni Platform enables companies to screen Alumni for their vacancies, while reducing the administrative burden over HYF coordinators and creating more jobs for skilled refugees & immigrants.

View project Contribute Front-end

Description

We have created a profile form where HackYourFuture alumni can show their coding skills and language proficiencies.

We enabled search function for companies to screen candidates behind a secure login.
See handover document for details.

It is build by HTML5, CSS3, Material UI, MongoDB, Express, NodeJS and ReactJS (MERN Stack). Both alumni and companies are authenticated and every API endpoint is protected using passport-JWT. You can contribute to the project by making pull requests.
